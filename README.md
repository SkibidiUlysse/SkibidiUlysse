## Projects

### [Grapheur](https://github.com/SkibidiUlysse/grapheur) — Interactive Math Plotter
> Type a function, watch it render. Smart parser, 6 themes (including a Miku theme), zoom & pan, multi-curve support — all in a single HTML file with zero dependencies.

### [Meteo Dashboard](https://github.com/SkibidiUlysse/meteo-dashboard) — Weather Intelligence
> Real-time weather for 3 cities via Open-Meteo API. Python backend generates bulletins, HTML5 Canvas frontend animates rain, snow & stars. No API key required.

### [L'Ascenseur Infernal](https://github.com/SkibidiUlysse/ascenseur-infernal) — Multi-Stage Escape Game
> 4-floor puzzle game: musical sequence memory, Greek cipher decoding, live maze navigation, and a full minesweeper board. Built with Python, Pygame & Tkinter.

### [Demineur](https://github.com/SkibidiUlysse/demineur) — Minesweeper Clone
> Classic minesweeper built from scratch with Pygame. Recursive flood-fill reveal, flag system, mine counter, win/loss detection.
